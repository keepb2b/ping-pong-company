FAQ
<Question 001> 依頼前に直接会って打ち合わせは可能ですか

可能です。

 

「こういうことがしたいと思っているけど本当にできるの？」

「CMSで管理しようとおもっているけど、大丈夫？」

といった疑問点がある場合、制作前にそちらの疑問を解消せずに制作を行うと、トラブルの元となってしまいます。

コーディング外注Coding ALIVEでは、お客様と実際にお会いし、実現しようとしている仕様・要件を確認して、発注いただく前からお打ち合わせを行うことができます。

例えばレスポンシブウェブを制作する場合ですと、CSSのレンタリングの仕組みを理解せずにデザインを作った場合、それを実現するために通常ではないコーディングを行わないと実現できなくなるといった可能性がございます。

「レスポンシブ対応のウェブを作らなくてはいけないけど、どうやってデザインを作ればいいか分からない」

「こういうところをWordPressのカスタムフィールドを使って更新出来るようにしたいけど、出来るのか分からない」

といった疑問がございましたら、是非お気軽にご相談くださいませ。

また、遠方で直接お会いできない場合でも、Zoom等での打ち合わせも対応いたします。

<Question 002> お客様のミーティングに一緒に参加してもらうことは可能ですか？

可能です。

 

デザイン作成前のお打ち合わせにも同行させて頂き、技術的な質問もその場でお答えさせていただきます。
複雑な仕様のプロジェクトでは、キックオフの段階からお打ち合わせに参加させて頂いた方がスムーズに進行できるかと思いますので、お気軽にご相談くださいませ。

<Question 003> レスポンシブデザインのコーディングはお願いできますか？

レスポンシブデザインのコーディングも可能です。

widthが100%の可変デザインのコーディングや、特定のwidthで固定され、スマートフォンサイト幅のブレイクポイントになった時にレイアウトを切り替える等の要望に合わせて、そのままコーディングを行います。

また、パソコン、タブレット、スマートフォンと切り替わる幅の指示があれば、そちらももちろん対応することが可能です。

<Question 004> ランディングページのコーディングはお願いできますか？

ランディングページのコーディングもお受けいたします。

ランディングページでは、制作する商品やキャンペーンをより魅力的に表現するために、アニメーションやパララックス表現を利用することがあるかと思いますが、そのような要望があった場合でも、まずはお気軽にご相談くださいませ。

ご相談がある場合はこちらからお願いします。

<Question 005> WordPressの構築は依頼できますか？

はい。もちろんご対応しております。
WordPressは実績が豊富にあり、カスタムフィールドの作成・各種カスタムのご要望もご依頼ください。

コーディング
ウェブページの構造を定義するためにHTML（HyperText Markup Language）を使ってコードを書くことです。HTMLコーディングでは、見出し、段落、リンク、画像、フォームなどの要素をマークアップし、ブラウザがそれらを適切に表示できるようにします。スタイルや機能はCSSやJavaScriptで追加されます。
CSS（Cascading Style Sheets）
HTMLで作成したウェブページのデザインやレイアウトを管理するためのスタイルシート言語です。CSSを使用することで、文字の色、フォント、レイアウト、背景、余白などを指定し、ページの見た目を調整できます。これにより、ウェブサイトの見た目を再現することが可能になります。
JavaScript
ウェブページに動的な機能やインタラクションを追加するためのプログラミング言語です。ブラウザ上で動作し、ユーザーの操作に応じてコンテンツの変更やアニメーション、データの送受信を実現します。HTMLやCSSと組み合わせて使用し、フォームの検証、ページのリアルタイム更新、アニメーション効果など、多様な機能を提供します。
jQuery
JavaScriptのライブラリで、DOM操作、イベント処理、アニメーション、Ajaxリクエストなどを簡単に実装できるツールです。クロスブラウザ対応を提供し、複雑なJavaScriptコードを簡潔に書けるようにします。2006年にリリースされ、幅広いブラウザでの互換性と豊富なプラグインが特徴です。
Webフォント
Webフォントは、ウェブページに表示する文字のスタイルを指定するためのフォントファイルで、サーバーから直接ブラウザに配信されます。これにより、ウェブデザインで利用できるフォントの選択肢が広がり、ユーザーの端末にインストールされていないフォントでも統一されたデザインを実現できます。無償で利用できるものと、利用するにあたってライセンスを買う必要があるものがあります。
メールフォーム
ユーザーがウェブサイト上で情報を入力し、メールとして送信するためのフォームです。通常、名前、メールアドレス、メッセージなどのフィールドが含まれ、送信ボタンを押すと、入力された情報が指定されたメールアドレスに送信されます。メールフォームは、ユーザーからの問い合わせやフィードバックを収集するために使用され、バックエンドでメールサーバーやサービスを介して処理されます。
サーバ
Web開発におけるサーバは、クライアント（ユーザーのブラウザなど）からのリクエストに応じて、ウェブページやデータを提供するシステムです。ウェブサイトやアプリケーションのコンテンツを保存し、ネットワークを通じてアクセス可能にします。HTTPリクエストを処理し、必要なデータやファイルをクライアントに返す役割を果たします。サーバは、専用のソフトウェア（Apache、Nginxなど）とハードウェアで構成されます。
ドメイン
インターネット上で特定のウェブサイトやサーバを識別するためのユニークなアドレスです。ドメイン名は、IPアドレスに代わる人間にとって覚えやすい文字列で、例えば「example.com」のような形式です。ユーザーがブラウザにドメイン名を入力すると、そのドメインに関連付けられたサーバにリクエストが送信され、ウェブサイトが表示されます。ドメイン名は、登録と管理が必要で、DNS（ドメインネームシステム）を通じてインターネット上で名前解決が行われます。
DNS
DNS設定とは、ドメインネームシステム（DNS）を用いてドメイン名とIPアドレスを関連付ける作業です。DNS設定により、ブラウザがドメイン名を使って正しいサーバにアクセスできるようになります。具体的には、AレコードでIPアドレスを設定したり、CNAMEレコードで別のドメインにリダイレクトしたりします。これにより、ユーザーがドメイン名を入力すると、正しいサーバーに接続し、ウェブサイトが表示されます。
CMS（コンテンツ管理システム）
管理画面を介して、ユーザーがウェブサイトを簡単に作成、管理、編集するためのソフトウェアです。テンプレートを使ってコンテンツを更新できるため、効率的なコンテンツ管理や誰でも更新できるサイト環境の構築が可能です。一般的に、記事、ページ、メディアの管理が行え、アクセス権の設定やバージョン管理などの機能も備えています。
WordPress
オープンソースのCMSで、主にブログやWebサイトの作成に使用します。基本は無料ですが、テーマやプラグインの追加に費用が発生することもあります。カスタマイズ性が高く、広く利用されているCMSです。
Movable Type
CMSで、企業向けの高機能ブログやサイト作成に特化しています。初期費用が高く、ライセンス料がかかります。静的な形式でデータを出力できるなど、セキュリティ面で優れています。
Shopify
クラウドベースのECプラットフォームで、オンラインストアの構築・運営が可能です。月額料金が発生しますが、サーバ管理の必要がない（=顧客情報を守ることをShopifyに委ねることができる）のと、決済の方法も比較的わかりやすく、使いやすいです。カスタマイズには限界があることもありますが、アプリを導入することで解決できる場合があります。
EC-Cube
オープンソースのECサイト構築ツールで、基本は無料です。カスタマイズや追加機能には費用がかかることがあります。高い柔軟性を持ちますが、運営には一定の技術力が必要です。サーバを用意し、そこに設置するため、顧客情報を守るための対策が必要です。
Makeshop
日本のECサイト構築サービスで、オンラインショップの設立が容易です。月額料金がかかりますが、豊富な機能とサポートが魅力です。カスタマイズには限界があることもありますが、アプリを導入することで解決できる場合があります。
ecforce
クラウド型ECプラットフォームで、豊富な機能を提供しています。導入コストは高めですが、デザインや機能の自由度が高いです。初期設定や運営には専門知識が必要です。
カラーミーショップ
日本のECサイト作成ツールで、低価格から利用可能です。使いやすさが特徴ですが、機能の拡張には追加料金が発生することもあります。サポートが充実しています。
Stripe
オンライン決済のサービスで、各種支払い処理を簡単に導入できます。手数料が発生しますが、技術的な導入がスムーズで、開発者に人気です。国際的なサポートも充実しています。
ヘッドレスCMS
フロントエンドとバックエンドを分離したCMSで、API経由でコンテンツを管理します。柔軟なデザインが可能ですが、実装には高度な技術が必要です。コストはCMSによって異なります。
React
UI構築のためのJavaScriptライブラリで、コンポーネントベースの開発が可能です。無料でオープンソースですが、学習曲線が急であることもあります。高いパフォーマンスと柔軟性が特徴です。
Next.js
Reactのフレームワークで、サーバーサイドレンダリングや静的サイト生成に対応しています。無料でオープンソースですが、最適化には技術的な理解が必要です。高いパフォーマンスが期待できます。
CSSアニメーション
Webデザインで使用されるスタイルシートの技術で、要素の動きや変化を定義します。ブラウザのサポートが広く、パフォーマンスへの影響が少ないですが、複雑なアニメーションには限界があります。
